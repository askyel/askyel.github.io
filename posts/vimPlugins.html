Vim Plugins

Pathogen | Syntastic | Airline | Fugitive | NERD Tree | Mark | Colorizer

Pathogen

Easily install and store plugins in subdirectory ~/.vim/bundle. https://github.com/tpope/vim-pathogen

Installation
$ mkdir -p ~/.vim/autoload ~/.vim/bundle
$ curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
vimrc Settings
execute pathogen#infect()
filetype plugin indent on
Install a Plugin
$ cd ~/.vim/bundle
$ git clone <GIT_REPO_LINK>

Syntastic

Runs file through external syntax checker and displays resulting errors to user. https://github.com/scrooloose/syntastic

vimrc Settings
set statusline+=%#warningmsg#
set statusline+=%{SyntasticStatuslineFlag()}
set statusline+=%*

let g:syntastic_always_populate_loc_list = 1
let g:syntastic_auto_loc_list = 1
let g:syntastic_check_on_open = 1
let g:syntastic_check_on_wq = 0
Commands
:SyntasticInfosee checkers available for current filetype
:SyntasticCheckperform manual check
:Errorsdisplay error window
:lcloseclose error window
:SyntasticResetclear error list
:SyntasticToggleModeswitch between active (check on writing buffer) and passive (manual check) modes
:lnextjump to next error
:lpreviousjump to previous error

Airline

Useful status/tabline for vim with customization and powerline symbol configuration. https://github.com/bling/vim-airline

vimrc Settings
let g:airline_powerline_fonts = 1
Commands
:AirlineTheme [theme-name]display or change the current theme
:AirlineToggletoggle between standard 'statusline' and airline

Fugitive

Advanced Git wrapper. https://github.com/tpope/vim-fugitive

Commands
:Git [args]run arbitrary git command
:Git! [args]run arbitrary git command; output captured in temp file
:Gstatus display git status in preview window
qclose status
rreload status
:Gcommit [args]run git commit; split window opened to write commit message
:Gmerge [args]call git merge
:Gpull [args]call git pull
:Gpush [args]call git push

NERD Tree

Explore and manipulate filesystem. https://github.com/scrooloose/nerdtree

Commands
:NERDTree [start-directory | bookmark]open NERD tree from directory or bookmark
:NERDTreeCloseclose NERD tree
:NERDTreeFindfind current file in tree
:NERDTreeCWDchange tree root to current directory
:Bookmark [name]bookmark current node as name
oopen file, directory, or bookmark

Mark

Highlights all occurrences of a word in file. Can highlight multiple identifiers in parallel. https://github.com/vim-scripts/Mark--Karkat

Commands
<Leader>mmark word under the cursor
<Leader>nclear mark under the cursor
:Markdisable all marks; automatically re-enabled on mark addition/deletion/search
:MarkClearclear all marks; cannot be undone
<Leader>*jump to next occurrence of current mark
<Leader>/jump to next occurrence of any mark
:MarkLoadrestore marks from previous session
:MarkSavesave currently defined marks

Colorizer

Colorizes text in #rrggbb, #rgb, or rgba(R,G,B, %) format. https://github.com/lilydjwg/colorizer

Commands
:ColorHighlightstart or update highlighting
:ColorClearclear all highlighting
:ColorToggletoggle highlighting
